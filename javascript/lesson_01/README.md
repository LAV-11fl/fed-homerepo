# JavaScript Lesson 01

Папка для першої домашньої роботи з курсу JavaScript (UA) 2025 - part 1.

## Примітки

- Назви папок уроків пишемо з двома цифрами: `lesson_01`, `lesson_02`, `lesson_03`.
- Рішення робимо простим і зрозумілим, як студентську домашню роботу.
- У коді додаємо короткі українські коментарі до важливих частин.
